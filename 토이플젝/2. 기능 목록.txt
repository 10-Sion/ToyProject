페이지 구성할 기능들 정리
cc
기능명: 검색 기능
	사용자는 지도 상에서 원하는 위치를 검색. 
	검색 결과로 해당 위치 주변의 음식점 정보를 보여줌

기능명: 음식점 정보 표시
	선택한 위치 주변의 음식점 정보를 보여줌.
	각 음식점의 이름, 주소, 평점 등의 정보가 포함

기능명: 음식점 선택
	사용자가 음식점을 선택하면 해당 음식점 주변의 카페 리스트를 가져와서 표시.

기능명: 카페 리스트 표시
	선택한 음식점 주변의 카페 정보를 리스트 형태로 보여줌.
	각 카페의 이름, 주소, 평점 등의 정보가 포함

기능명: 음식점 및 카페 선택 완료
	사용자가 원하는 음식점과 카페를 선택하면 
	선택한 내용을 일정으로 만들어서 저장할 수 있는 기능

기능명: 일정 생성
	사용자가 선택한 음식점과 카페를 포함한 일정을 생성.
	일정의 제목, 날짜, 시간 등을 설정

기능명: SNS 공유
	생성된 일정을 소셜 미디어 플랫폼에 공유할 수 있는 기능.

기능명: 사용자 계정 관리
	사용자는 회원 가입과 로그인을 통해 자신의 일정을 관리하고 공유.

기능명: 위치 정보 표시
	사용자가 검색한 위치를 지도 위에 마커로 표시하여 시각적으로 확인.

기능명: 반응형 디자인
	레이아웃을 최적화하여 반응형 디자인을 구현.
