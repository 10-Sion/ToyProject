프론트

1. 첫 화면
: 중앙 상단 제목, 중앙 중간 사진 (양 옆 슬라이드), 아래 소개 글, 아래 시작버튼 (크게) / 오른쪽 위 상단 로그인, 회원가입

2. 회원가입 클릭
: 회원가입 양식 화면 출력 후 오른쪽 상단 로그인 클릭 -> 로그인 화면 출력
로그인 화면 밑에도 회원가입 버튼 -> 회원가입 화면 출력

3. 로그인 후, 지도화면 및 검색 구간 (로그인했을 시 오른쪽 상단 아이디명, 로그아웃,마이페이지 추가)
: 중앙 상단 검색버튼 및 연관검색어 기능 추가,
중간위치 지도 (크게)

4. 검색버튼 클릭 및 엔터 누를 시,
: 중앙 지도 화면에 아이콘 출력. 하단에 (전체/한/일/양/중/기타) 카테고리 출력

5. 아이콘 클릭 시, 음식점 정보 표시 (이름,주소,평점 등) / 오른쪽 화면 출력 (슬라이드 - 상세보기 및 접기 가능)
-> 찜 선택 (마이페이지 추가) / 마이페이지 : 찜 목록, 저장 목록
고정 선택 시 (오른쪽 상단 고정). 아래 근방 카페 출력

5-1. 카테고리 클릭 시
: 검색한 근방 음식점 카테고리 별로 출력 (하단)
음식점 클릭 시, 중앙 지도 화면에서 아이콘으로 위치 표시
-> 찜 선택 (마이페이지 추가) / 마이페이지 : 찜 목록, 저장 목록
고정 선택 시 (오른쪽 상단 고정). 아래 근방 카페 출력

6. 카페까지 선택 완료 후
오른쪽 화면에서 바로 메모할 수 있게 메모 기능 추가 및 일정 추가 가능
(아래 저장 버튼)

7. 저장 후 저장 버튼 옆 바로 공유 버튼 생성
: 카카오링크 전송 가능
(마이페이지 저장 목록에서도 전송 가능)

-------------------------------------------------------------
로그인 없이 바로 시작버튼 누를 시,
-> 2번 생략 및 모든 화면 오른쪽 상단 아이디명, 로그아웃, 마이페이지 없음

카카오 링크 공유 가능하지만 저장, 일정(메모)추가 기능 이용 불가.
