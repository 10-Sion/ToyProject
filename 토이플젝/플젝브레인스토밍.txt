구현 목표 : 지도 상 특정 위치(음식점 등)의 선택 시 근방 일정 거리의 
              카페를 리스트로 보여주고,  음식점과 카페의 정보를 sns를 
              통해 공유할 수 있게 하는 것

1. 웹 페이지의 실행
2. 초기 소개화면 : 페이지 소개글과 시작 버튼
3. 시작 버튼 클릭 시 지도 화면 이동 : 지도 위 검색 기능
4. 주소의 검색 시 근처의 음식점을 보여줌
5. 음식점의 선택 시 근처 반경의 카페 리스트를 보여줌
6. 음식점과 카페 선택이 완료되면 이를 일정으로 만들어 sns상 공유

	위치의 특정법? - 주소 검색, GPS 기반, 지도상 클릭,
		         장소의 검색, GPS 트래킹(실시간)
	     실현 가능? - 주소 검색, GPS, 지도상 클릭, 장소 검색

1. 장소의 입력이나 지도상 위치 클릭(지도 바로 표시됨)의 두 방법 선택지
